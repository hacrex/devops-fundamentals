# Phase 7: Reliability Engineering

This phase covers SRE practices, incident management, and capacity planning.

## Topics Covered

1. **SLA/SLO/SLI** - Service level objectives and indicators
2. **Error Budgets** - Balancing reliability and velocity
3. **Incident Management** - Response, postmortems, learning
4. **Capacity Planning** - Scaling strategies, resource management

## Learning Path
```
SLA/SLO/SLI → Error Budgets → Incident Management → Capacity Planning
```

## Prerequisites
- Complete Phase 6 (CI/CD & IaC)
- Production operations experience

## Topics

### [Topic 1: SLA/SLO/SLI](./topic-01-sla-slo-sli.md)
Learn to define and measure service level objectives and indicators.

### [Topic 2: Error Budgets](./topic-02-error-budgets.md)
Understand error budgets and burn rate monitoring for release decisions.

### [Topic 3: Incident Management](./topic-03-incident-management.md)
Master incident response, communication, and blameless postmortems.

### [Topic 4: Capacity Planning](./topic-04-capacity-planning.md)
Plan and scale infrastructure to meet current and future demand.

## Hands-On Labs

1. Define SLOs for a service
2. Calculate error budgets
3. Create incident runbooks
4. Configure auto-scaling policies
