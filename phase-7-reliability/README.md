# Phase 7: Reliability Engineering

This phase covers SRE practices, incident management, and capacity planning.

## Topics Covered

1. **SLA/SLO/SLI** - Service level objectives and indicators
2. **Error Budgets** - Balancing reliability and velocity
3. **Incident Management** - Response, postmortems, learning
4. **Capacity Planning** - Scaling strategies, resource management

## Learning Path
```
SLA/SLO/SLI → Error Budgets → Incident Management → Capacity Planning
```

## Prerequisites
- Complete Phase 6 (CI/CD & IaC)
- Production operations experience
