# Phase 6: CI/CD & Infrastructure as Code

This phase covers continuous integration, deployment, and infrastructure automation.

## Topics Covered

1. **CI/CD Concepts** - Pipelines, stages, best practices
2. **GitHub Actions** - Workflow automation
3. **ArgoCD** - GitOps continuous delivery
4. **Terraform** - Infrastructure as Code

## Learning Path
```
CI/CD Concepts → GitHub Actions → ArgoCD → Terraform
```

## Prerequisites
- Complete Phase 5 (Traffic & Resilience)
- Git fundamentals

## Topics

### [Topic 1: CI/CD Concepts](./topic-01-cicd-concepts.md)
Learn the fundamentals of continuous integration and delivery pipelines.

### [Topic 2: GitHub Actions](./topic-02-github-actions.md)
Automate your build, test, and deployment workflows with GitHub Actions.

### [Topic 3: ArgoCD](./topic-03-argocd.md)
Implement GitOps continuous delivery for Kubernetes with ArgoCD.

### [Topic 4: Terraform](./topic-04-terraform.md)
Define and manage infrastructure as code using Terraform.

## Hands-On Labs

1. Create CI/CD pipeline with GitHub Actions
2. Deploy application using ArgoCD
3. Provision infrastructure with Terraform
4. Implement GitOps workflow
